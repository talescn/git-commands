## Como clonar um repositóio

Passo a passo:

```bash
# Navegando até o disco C
cd /c

# Criando uma pasta
mkdir minha-pasta

# Entrando na pasta
cd minha-pasta 

# Clonando o repositório
git clone <link-repositorio>

# Acessando o repositório
cd <path-repositorio>
```
